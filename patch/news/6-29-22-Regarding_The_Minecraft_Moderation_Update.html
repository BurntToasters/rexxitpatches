REXXIT: Regarding the minecraft moderation update [6-29-22]

Hi there! Hopefully most people are aware of the MC Moderation update (see here)
And whilst this is a horrible update brougt to the community and completely removes the ability for one to moderate their own MC server freely, there seems to be work around for this in
order to restore the server owners ability to moderate their own. Disabling message key signing (which proves authenticity that a users message did come from them and not an unauthorized source)
will actually removed MC's ability (as of now) to be able to report someone as there is no way to prove a message came from the actual person. Once 1.19.1 comes out which is going to be
the first update which introduces this, I will be modifying the server to disabled message signing. Since this is a closed and private server, disabling this feature wont do anything
at all, but no one knows for how long that this will work. I will keep people updated with this issue and we will go from there.
